Digital Scams & Cyber Awareness
Overview

This project is a web-based platform to educate people about digital scams and promote cyber awareness. It aims to inform users about different types of scams, prevention tips, and helplines, helping them stay safe online and offline.

Features

Interactive Navigation Menu: Easy access to Home, About, Types of Scams, Prevention Tips, and Help sections.

Detailed Scam Information: Covers various scam types:

Online Scams

Phone & Message Scams

Financial Scams

Identity Scams

Offline Scams

Specialized/Modern Scams

Scam Recognition Tips: Helps users identify suspicious activities and avoid falling victim.

Prevention Guidelines: Lists do’s and don’ts for staying safe.

Report & Help Section: Provides helpline numbers and reporting instructions.

Responsive Design: Works on both desktop and mobile devices.

How to Use

Clone this repository:

git clone https://github.com/Yamunadevi006/Scam-awareness.git


Open index.html in a web browser.

Navigate through the menu to explore scam types, prevention tips, and help resources.

Click the View More buttons to access detailed pages for each scam type.
